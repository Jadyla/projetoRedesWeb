Sistema Operacional

O sensor Sistema Operacional tem como objetivo mostrar as informações do sistema operacional da máquina em consulta. Para isso foi utilizado o oid sysDescr. Para realizar a consulta basta informar um IP válido, e que nesta máquina o protocolo SNMP esteja ativo. Após a consulta serão informados o sistema operacional, hardware e software do equipamento.
